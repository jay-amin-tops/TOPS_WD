Psuedo Class Selector:

- Defination: It is used to give style to the class properties of a tag/class. eg. hover, active, link, etc.

- It is denoted by class_name/tag:class_Selector

- There are two types of psuedo class:

1. Link Psuedo Class: It gives styling to static properties of tags which are based on user previous interactions.

-- Types of Link Pusedo Class:

i. Link: It defines the styling of anchor tag link which not been visited before.

-Ex:-
Default: Home
Link selector: Home

ii. Visited: It defines the styling of anchor tag link which is visited atleast onece

-Ex:-
Default: Home
Visited selector: Home

2. Dynamic Pusedo Classes: It gives styling to dynamic properties of tags which are based on user interactions.

-- Types of Dynamic Psuedo-class:

i. Active: It give styling to link when its active pressed.

-Ex:-
Default: Home
Active Selector: Home

ii. Hover: It gives styling to an tag whem its hovered over.

-Ex:- Default: Home
Hover Selector: Home

iii. Focus: It gives styling to an tag when its focused.

-Ex:- Default: Home
Focus: Home

iv. first-child/nth-child/last-child: To give CSS to specific tags without using class or id.

-Ex:-
abc
mno
pqr

Psuedo-elements Selectors:

- Defination: It is used to give style to the element properties of a tag/class.

- It is denoted by class_name/tag::elements_Selector

-- Types of Elements Selector:

1. fist-line: It gives style to first line of the tag.

-Ex:-

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Eligendi blanditiis molestiae eos porro esse beatae sapiente veritatis reprehenderit explicabo impedit? Dignissimos velit deserunt delectus eos est, labore incidunt! Aspernatur, rerum.

2. firt-letter: It gives style to first letter of the tag.

-Ex:-

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Vero cum hic quia itaque alias facilis illo non dicta et. Aliquam nobis vel minus aperiam libero labore consectetur, fuga dolorum ipsum?

3. Selection: It gives style to select area of the tag.

-Ex:-

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Adipisci harum eos exercitationem, repudiandae aspernatur, ut minus doloribus ex officia iste ipsa magnam maiores. Aliquid repudiandae odit, delectus necessitatibus neque molestiae.

4. Marker: It gives style to bulltes and numbers .

-Ex:-

  • one
  • two
  • three
